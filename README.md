# maso.fun

> 记录自己学习记录点滴。

## 简介

个人网站 [maso.fun](https://maso.fun)，纯静态 HTML 页面，作为导航主页和学习记录入口。

## 内容

| 页面                  | 说明 |
|---------------------|------|
| `index.html`        | 主站首页，展示个人资料与常用链接 |
| `mirror/index.html` | 魔法镜 — 调用摄像头实现动漫风格梳妆镜，支持滤镜切换、拍照保存 |

## 本地运行

任意静态文件服务器均可.
